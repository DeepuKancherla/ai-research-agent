# 🤖 AI Research Agent

## 🚀 Overview
An AI-powered research assistant that fetches data from web sources and generates structured explanations with source references.

## ✨ Features
- 🔍 Smart topic search
- 📄 Extracts real data from Wikipedia
- 🧠 Generates clean explanations
- 🔗 Provides source links
- 🌐 Streamlit web interface

## 🛠 Tech Stack
- Python
- BeautifulSoup
- Streamlit

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
